# 汎用 Fable 5 システムプロンプト

[![ライセンス：MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
![互換性](https://img.shields.io/badge/Compatibility-Claude%20%7C%20ChatGPT%20%7C%20Gemini%20%7C%20DeepSeek%20%7C%20Qwen-blue)

[English](README.md) | [简体中文](README.zh-CN.md) | [日本語](README.ja.md)

Anthropic が公開している **Claude Fable 5 / Mythos 5** システムプロンプトを、ベンダーに依存しない形に改変したものです。

元のプロンプトには、XML の規約、ツールインターフェース、製品固有の動作、内部実装のガイダンスなど、Claude の本番環境専用の基盤が大量に含まれています。これらの内容は Claude には適していますが、ほかの最先端 LLM で再利用する場合、その多くはほとんど、あるいはまったく役に立ちません。

このリポジトリでは、そうした実装の詳細を取り除きつつ、現代のモデル全般に適用できる行動ガイダンスを保持しています。

---

## 目標

* コンテキストを効率的に使えるよう、プロンプトのサイズを削減する。
* Claude 固有の実装詳細を取り除く。
* 影響度の高い行動ガイダンスを保持する。
* 主要な最先端 LLM との互換性を維持する。

---

## バージョン

| ファイル                | 用途                                                                                     |
| ----------------------- | ---------------------------------------------------------------------------------------- |
| `universal/core.md`     | 最小限のトークン使用量で、行動に関する中核原則を提供します。                             |
| `universal/balanced.md` | ほとんどのユースケースに推奨されます。プロンプトのサイズと行動の網羅性を両立します。     |
| `universal/complete.md` | モデル非依存性を維持しながら、ベンダーに依存しない行動ガイダンスをほぼすべて保持します。 |

---

## 取り除いたもの

このプロジェクトでは、以下のような Claude 固有の実装を意図的に除外しています。

* XML ラッパーと書式規約
* MCP およびツール固有の基盤
* 製品固有の指示
* 内部 UI の動作
* ストレージと成果物の管理
* 環境固有の実装詳細

---

## 保持したもの

選別後のプロンプトには、以下のような行動ガイダンスが保持されています。

* 応答前にユーザーの意図を理解する
* 不確実性を正直に扱う
* 明確なコミュニケーションと自然な文章
* 高品質なコード生成
* トレードオフ分析を含む実用的な推奨
* 最終回答の前に検証する
* 客観的に誤りを訂正する
* 不要なメタコメントを避ける

---

## 互換性

これらのプロンプトは、以下を含む現代の最先端モデルで良好に動作するよう設計されています。

* Claude
* ChatGPT
* Gemini
* DeepSeek
* Qwen
* Grok
* GLM
* その他の指示に従う LLM

---

## 使い方

### API プラットフォーム

目的のプロンプトの内容を、モデルのシステムプロンプトまたはシステム指示のフィールドに貼り付けます。

### 一般ユーザー向けインターフェース

対応している場合、これらのプロンプトはカスタム指示、カスタム GPT のシステムプロンプト、Gem、または同等の機能としても利用できます。

---

## 方針

このプロジェクトは、Anthropic の元のシステムプロンプトを置き換えたり、それを改善したと主張したりすることを意図していません。

代わりに、別の問いを提示します。

> **Claude 固有の実装詳細を取り除いた後も、そのシステムプロンプトに含まれるどの行動原則が価値を持ち続けるのか？**

その結果、プラットフォーム固有の機能ではなく、移植性に最適化された一連のプロンプトが得られます。
