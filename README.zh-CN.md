# 通用 Fable 5 系统提示词

[![许可证：MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
![兼容性](https://img.shields.io/badge/Compatibility-Claude%20%7C%20ChatGPT%20%7C%20Gemini%20%7C%20DeepSeek%20%7C%20Qwen-blue)

[English](README.md) | [简体中文](README.zh-CN.md) | [日本語](README.ja.md)

这是对 Anthropic 公开提供的 **Claude Fable 5 / Mythos 5** 系统提示词所做的厂商中立改编。

原始提示词包含大量专用于 Claude 生产环境的基础设施，包括 XML 约定、工具接口、产品特定行为和内部实现指南。虽然这些内容适用于 Claude，但在其他前沿大语言模型中复用时，其中许多内容几乎没有或完全没有帮助。

本仓库过滤了这些实现细节，同时保留了可在现代模型间广泛适用的行为指南。

---

## 目标

* 缩减提示词大小，提高上下文效率。
* 移除 Claude 特有的实现细节。
* 保留影响较大的行为指南。
* 保持与主流前沿大语言模型的兼容性。

---

## 版本

| 文件                    | 用途                                                                               |
| ----------------------- | ---------------------------------------------------------------------------------- |
| `universal/core.md`     | 以最少的 token 用量提供核心行为原则。                                              |
| `universal/balanced.md` | 推荐用于大多数场景，在提示词大小和行为覆盖范围之间取得平衡。                       |
| `universal/complete.md` | 保留几乎所有厂商中立的行为指南，同时保持模型无关性。                               |

---

## 移除了什么

本项目有意排除了 Claude 特有的实现内容，例如：

* XML 包装器和格式约定
* MCP 及工具特定的基础设施
* 产品特定的指令
* 内部 UI 行为
* 存储和产物管理
* 环境特定的实现细节

---

## 保留了什么

经过筛选的提示词保留了以下行为指南：

* 回复前理解用户意图
* 如实处理不确定性
* 清晰沟通和自然写作
* 生成高质量代码
* 提供包含取舍分析的实用建议
* 在最终作答前进行验证
* 客观纠正错误
* 避免不必要的元评论

---

## 兼容性

这些提示词旨在良好适配现代前沿模型，包括：

* Claude
* ChatGPT
* Gemini
* DeepSeek
* Qwen
* Grok
* GLM
* 其他遵循指令的大语言模型

---

## 用法

### API 平台

将所需提示词的内容粘贴到模型的系统提示词或系统指令字段中。

### 面向消费者的界面

如果平台支持，也可以将这些提示词用作自定义指令、自定义 GPT 系统提示词、Gem 或同类功能。

---

## 理念

本项目无意取代 Anthropic 的原始系统提示词，也不声称对其有所改进。

相反，它提出了一个不同的问题：

> **移除 Claude 特有的实现细节后，其系统提示词中的哪些行为原则仍然有价值？**

最终得到的是一组针对可移植性而非平台特定功能优化的提示词。
